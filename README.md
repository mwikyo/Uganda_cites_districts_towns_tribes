# Uganda_cites_districts_towns_tribes
A full list of all districts, cities, municipalities/towns and tribes in Uganda.

Each list is provided in three file types: csv, yaml and json.

Example:
`- district: Abim
    region: North
    sub-region: Karamoja`

or 
`{
        "district": "Abim",
        "region": "North",
        "sub-region": "Karamoja"
  }`

  or
  `"Abim","North","Karamoja"`
